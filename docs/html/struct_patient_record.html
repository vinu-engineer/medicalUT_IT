Detailed Description

Complete patient record including demographics and vital sign history.

The readings array acts as a sequential log: index 0 holds the first reading, index reading_count-1 holds the most recent. The record is considered full when reading_count == MAX_READINGS.

Note
No dynamic memory is used. All storage is stack/static allocated.
